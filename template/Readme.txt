To run just the simulation the syntax is:
./a.out <num_threads> <threshold> 0 <num_ticks>

To run the simulation and write the final universe state to a file, the syntax is:
./a.out <num_threads> <threshold> 1 <num_ticks> <universeFilename>

To run the simulation and write a heatmap of the final universe state to a file, the syntax is:
./a.out <num_threads> <threshold> 2 <num_ticks> <universeFilename> <heatmapFilename>
